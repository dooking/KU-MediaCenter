# KU-MediaCenter🎬
>고려대학교 교육매체실은 일반적인 행사기록, 학교 홍보 및 각종 정책 소개영상 제작을 지속적으로 늘려왔습니다. 이러한 노력을 통해 교내외 유관부서 및 기관들과 협력하여 영상매체를 통한 고려대학교의 교육과 연구 지원은 물론 대외 홍보와 학교 이미지 제고에 직간접적으로 기여하고 있습니다.
